## Features

### Transfer To

Allows transferring money from one account to another.  
Just provide the destination username and the amount to transfer.

### Request Loan

Requests a loan.  
To be approved, the account must have at least one deposit that is equal to or greater than 10% of the requested loan amount.  
If this requirement is not met, the request is denied.

### Close Account

Closes the current account.  
You must enter the username and PIN of the logged-in account to confirm.
